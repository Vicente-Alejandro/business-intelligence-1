# 📊 Business Intelligence Repository

A beginner-friendly introduction to **Business Intelligence** combining databases, Python, and PowerBI. This repository showcases a simple end-to-end BI workflow with practical examples.

---

## 🎯 Overview

This project demonstrates the fundamentals of Business Intelligence by creating a simple data pipeline:

**Database → Python Extract & Transform → PowerBI Visualization**

---

## 📦 Technologies Used

- **Database**: SQL (SQLite/MySQL as you wish)
- **Data Processing**: Python
- **Visualization**: Microsoft PowerBI
- **Tools**: Jupyter Notebooks, VS Code

---

## 🚀 Quick Start

### Prerequisites

- Python 3.8+
- PowerBI Desktop
- SQL database (SQLite included by default)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Vicente-Alejandro/business-intelligence-1.git
cd business-intelligence-1
```

2. Install Python dependencies:
```bash
pip install -r requirements.txt
```

## 📚 Simple Exercise: Sales Analysis

### Objective
Analyze sales data and create a basic dashboard showing:
- Total Sales by Region
- Monthly Revenue Trend
- Top Products

### Data Flow
1. **Database**: Sales transactions table
2. **Python**: Extract, filter, and aggregate data using Pandas
3. **PowerBI**: Create interactive visualizations and KPIs


## 📊 Key Concepts Covered

- ✅ Data Extraction from databases
- ✅ Data Transformation and cleaning
- ✅ Data aggregation and analysis
- ✅ Dashboard creation and visualization
- ✅ KPI tracking and reporting
